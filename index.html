Our Current Workout Plan: PPL (Push Pull Legs)

PUSH

Chest:

  • Incline Smith Machine Bench Press | 3 sets, 6-8 reps
  • Seated Chest Press Machine | 3 sets, 6-8 reps
  • High Cable Chest Fly | 3 sets, 6-8 reps
  • Pec Deck Machine | 3 sets, 6-8 reps

Triceps:

  • Cable Bar Tricep Pushdown | 3 sets, 8-10 reps
  • Cable Bar Overhead Tricep Extension | 3 sets, 8-10 reps

Shoulders:

  • Machine Shoulder Press | 3 sets, 6-8 reps
  • Dumbbell Lateral Raises | 3 sets, 8-10 reps
  • Cable Rope Front Raises | 3 sets, 8-10 reps

PULL

Back:

  • Wide-Grip Lat Pulldown | 3 sets, 6-8 reps
  • V-Handle Cable Seated Row | 3 sets, 6-8 reps
  • Machine Row | 3 sets, 6-8 reps
  • Cable Bar Straight Arm Pulldown | 3 sets, 8-10 reps
  • Machine Rear Delt Reverse Flys | 3 sets, 8-10 reps

Biceps:

  • Dumbbell Incline Bicep Curls | 3 sets, 8-10 reps
  • Machine Preacher Curls | 3 sets, 8-10 reps
  • Cable Rope Hammer Curls | 3 sets, 8-10 reps

LEGS

Glutes / Hamstrings:

  • Prone Leg Curl Machine | 3 sets, 6-8 reps
  • (Smith) Machine Hip Thrusts | 3 sets, 6-8 reps

Quads:

  • Barbell Squats | 3 sets, 6-8 reps
  • Leg Press Machine | 3 sets, 6-8 reps
  • Leg Extension Machine | 3 sets, 8-10 reps

Abductors / Adductors:

  • Abductor Machine | 3 sets, 8-10 reps
  • Adductor Machine | 3 sets, 8-10 reps

Start Now!!

It is recommended to take between 1-2 minutes of rest in between sets.
